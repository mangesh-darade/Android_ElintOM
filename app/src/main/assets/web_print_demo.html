‚ú® Features

Available Print Features:

  • Floating Print Button - Always visible print button
  • Keyboard Shortcut - Press Ctrl+P to print
  • Context Menu - Right-click to print selected text
  • Printer Selection - Choose from connected printers
  • Print Settings - Adjust paper width, line spacing, font size
  • Auto-Detection - Automatically finds connected printers

üîß Integration

How to integrate in your web pages:

1. Include the ElintPOS printer library:

<script src="file:///android_asset/elintpos-printer.js"></script>

2. The library automatically adds print functionality to your page

3. Use JavaScript to print content programmatically:

// Print specific content elintposPrinter.printContent('Your content here'); // Print with specific printer elintposPrinter.printContent('Your content here', 'usb'); // Check printer status elintposPrinter.getAvailablePrinters().then(printers => { console.log('Available printers:', printers); });
